# Dead By Daylight Knowledge Repository

A comprehensive collection of Dead By Daylight knowledge, strategies, and analysis for both new and experienced players.

## Repository Structure

### Core Documents

1. **[Key Concepts & Game Mechanics](key_concepts.v1.md)**
   - Overview of Dead By Daylight
   - Core gameplay mechanics
   - Character roles and objectives
   - Progression systems
   - Basic strategies

2. **[Character Profiles](character_profiles.md)**
   - Detailed Survivor profiles and backstories
   - Killer powers and abilities
   - Unique perks for each character
   - Character selection strategies

3. **[Advanced Strategies & Meta Analysis](advanced_strategies.md)**
   - Advanced Survivor techniques
   - Killer pressure management
   - Current meta trends
   - Map-specific strategies
   - Ranked play optimization

4. **[Perk Analysis & Meta Builds](perk_analysis.md)**
   - Comprehensive perk breakdowns
   - Meta builds for both sides
   - Perk tier lists
   - Patch history impact
   - Counter-meta strategies

5. **[Map Knowledge & Environmental Strategies](map_knowledge.md)**
   - Detailed map breakdowns
   - Environmental elements and interactions
   - Map-specific meta analysis
   - Advanced tile recognition
   - Spawn prediction techniques

## Learning Path

### For New Players
1. Start with **Key Concepts & Game Mechanics**
2. Read **Character Profiles** to understand roles
3. Practice basic strategies from **Advanced Strategies**
4. Learn fundamental perk usage from **Perk Analysis**

### For Intermediate Players
1. Study **Advanced Strategies** for technique improvement
2. Master **Perk Analysis** for optimal builds
3. Learn **Map Knowledge** for environmental awareness
4. Practice character-specific strategies from **Character Profiles**

### For Advanced Players
1. Refine techniques from **Advanced Strategies**
2. Optimize builds using **Perk Analysis** meta knowledge
3. Master map control through **Map Knowledge**
4. Develop counter-strategies for current meta
